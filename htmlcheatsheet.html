Attributes

This example can be used to define each individual attribute.

href Attribute src Attribute width and height Attribute alt Attribute style Attribute lang attribute
The tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to The tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed The tag should also contain the width and height attributes, which specify the width and height of the image (in pixels) The alt attribute is required when inserting an img tag. It specifies an alternate text description for an image if the image cannot be displayed. The style attribute is used when adding styles to elements, like font, size, color, etc. The lang attribute should be included inside the html tag. This declares the language of the webpage
