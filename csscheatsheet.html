CSS Cheat Sheet



CSS Selectors

Universal Selector(*) Element Selector
Selects all elements on the page and applies the same style universally. For example, setting the font color for every element. Targets all elements of a specific type, such as paragraphs or headers. For example, setting a common font size for all paragraphs.

Text Styles

Color Text Alignment Text Decoration Text Spacing
The color property is used to set the color of the text. The color is specified by: a color name - like "red" a HEX value - like "#ff0000" an RGB value - like "rgb(255,0,0)" The text-align property is used to set the horizontal alignment of a text. Text can be aligned to the center, right, left or justified. The text-decoration-line property is used to add a decoration line to text. Values can be combined; like overline and underline to display lines above and below text. The letter-spacing property increases or decreases the space between characters in a text.

CSS Syntax

Selector and Declaration
The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons. Each declaration includes a CSS property name and a value, separated by a colon. Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.
